If you want to use Nèflody, you'll need a developer account on Spotify.

Here's how you can get started:

Visit the Spotify Developer Dashboard

Log in or sign up for a developer account if you haven't already.

Once logged in, create a new app. This will be necessary for integrating Spotify's features with Nèflody.

After creating your app, go to the app settings and change the redirect URI to:
